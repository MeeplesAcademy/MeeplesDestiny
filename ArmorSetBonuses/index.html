Instructions

Welcome to Armor Set Bonuses! Just a little tool I made because I needed it, then thought to share it with the community. Using this you can find the armor sets that synergize with potential builds. Here are some functions and ways you can use it:

Control Panel: determine how you want to see things.

  • Choose "Show Only" (default) to see only what you're looking for —OR— "Highlight" to see what stands out against all the sets.
  • Match Any to see all your selections together —OR— Match All to only see exactly what you're choosing. (If a set doesn't include both/all of your filtering selections, it won't show.)

Filtering Options: what are you looking for?

  • String search bar — type in pretty much anything and the sets with matching text strings will show.
  • Source filters — click these to see where sets can be acquired.
  • Elemental keyword filters — these help find subclass-oriented keywords (The Select All is so you can see literally everything highlighted for fun.)
  • Miscellaneous keywords — this collapsible section helps find Benefits, Weapon types, Triggers, and some extra Combat-flavored text in sets.

HAVE FUN. Hope it's helpful. And check out the great tools on the right: D2 Armor Calculator will help you design & build and figure out which armor to chase. Destiny Item Manager (DIM) will help you design & build loadouts from your actual inventory and save them for game use.
